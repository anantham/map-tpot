Design Decisions

Navigation
← Back link at top. Community sibling nav at bottom. Member handles link to /?handle=X. Share button copies /?community=slug.
Tweet Cards
Community-color left border for top tweet per member. Grey for rest. "↗ view on X" links to x.com/user/status/id. Shows type badge (thread/reply/tweet).
Data
Top 5 members by NMF weight, 3-5 tweets each. Remaining members: handle + bio only. All from enriched data.json export.
URL
/?community=llm-whisperers-ml-tinkerers — slug generated from community name at export time. Homepage showcase tags become clickable links.
